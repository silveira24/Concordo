\doxysubsection{Descrição detalhada}
Classe que representa o sistema Concordo. 

Esta classe gerencia os usuários, servidores e interações do sistema Concordo. Ela permite criar usuários, criar servidores, gerenciar participantes, realizar login e muito mais.
